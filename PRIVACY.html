Chrome Extension

Privacy Policy

Ask Your Past  ·  Last updated: March 17, 2026

The short version: Everything stays on your device. The developer collects no data — no analytics, no telemetry, no backend.

Overview

Ask Your Past is a Chrome extension that lets you search and chat with your own browsing history. It is built with privacy as a first principle. No data is ever sent to the developer, and no backend server exists.

What Data the Developer Collects

None. There is no analytics, no telemetry, no crash reporting, and no server operated by this extension.

What Data the Extension Uses (Locally)

All data stays on your device, stored in your browser's local IndexedDB.

Browsing History

  • The extension reads the title and URL of pages you visit.
  • This data is stored locally in IndexedDB — never transmitted anywhere.

Page Content

  • The background service worker may read the title and meta description of pages to improve search quality.
  • No full page content, passwords, form data, or authenticated content is stored.

Embeddings

  • Titles and URLs are converted into vector embeddings via the OpenAI API.
  • These embeddings are stored locally in IndexedDB alongside the original text.

Your OpenAI API Key

  • You provide your own OpenAI API key in the extension settings.
  • It is stored in chrome.storage.local — local to your browser, not synced to any cloud by this extension.
  • It is used only to make requests from your browser directly to api.openai.com (embeddings and chat).
  • The developer never sees, receives, or stores your API key.

Third-Party Services

The only external service this extension contacts is OpenAI (api.openai.com), and only when you trigger an embedding or chat action. Your use of OpenAI is governed by OpenAI's Privacy Policy.

Data Deletion

All extension data lives in your browser's IndexedDB. Delete it any time by uninstalling the extension or clearing its site data in Chrome settings. There is no remote copy to request deletion of.

Children's Privacy

This extension is not directed at children under 13 and does not knowingly collect any information from children.

Changes to This Policy

Any changes will be published in the extension's repository with a new effective date. Continued use after changes constitutes acceptance.
